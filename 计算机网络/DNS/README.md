# dns

客户端向DNS服务器查询域名，一般返回的内容都不超过512字节，用UDP传输即可。

不过，UDP服务，则需要超时重传机制

不用经过TCP三次握手，这样DNS服务器负载更低，响应更快。
虽然从理论上说，客户端也可以指定向DNS服务器查询的时候使用TCP，但事实上，很多DNS服务器进行配置的时候，仅支持UDP查询包。
